生活日记-20240401

发表于2024-04-01|更新于2024-09-26|生活
|字数总计:1k|阅读时长:3分钟|阅读量:

今天愚人节，bilibili 每年一度的小会员又出现了，才突然意识到今天就是四月一号，三月份已经结束了。

24 年已经过去四分之一了耶，总结下三月份做了哪些事吧，警告下自己，别偷懒。

开源项目

三月初，创建了新项目 iw-mixes ，这是去年年终总结的未来期望之一。没错，跳过了找工作，先把这个计划做起来了，原因就是工作躺平，不想去接触新环境、新变化，其次就是一直心心念念着开源项目，毕竟作为程序员，做一个属于自己的开源项目可太酷了，不得不感慨一下兴趣真的是一个很好的驱动力。

项目在三月份主要实现了一个简单的登录认证服务，基于 SpringSecurity 实现表单登录、OAuth2-GitHub 登录功能。创建了一个笔记服务，还未动工，在犹豫是先有 web 界面，还是直接撸起袖子写接口。

做开源项目的一点小感想：登上 GitHub Trending 的项目真牛！当自己真正想从 0 开始搭建一个项目时，才发现有多麻烦，想要做到 1.0 版本发布，真不是一时半会的事（大佬除外）。

因为 iw-mixes 是一个供自己学习的融合项目，所以项目特性是想一出是一出的...做出来的功能又不希望是一个简单呆板、没有任何头绪的东西，因此有时候写着写着就发现不知道后面要写啥了。为此，我做了一个 futures 文档，将未来大致的功能特性先记上，再将平时脑洞大开的小功能写在对应的大纲下，这样每次动工时，可以很快知道接下来要干啥，并且尽量将一个小功能的工期缩短，避免因为其他原因影响开发节奏。

读书 - 深入浅出Spring Security

今年开年第一个月读完了一本书，书是 21 年买的，松哥（江南一点雨）写的《深入浅出Spring Security》，封面图如下：

当时还是 SpringBoot 2 + SpringSecurity 5 的天下，如今已经是 SpringBoot 3 + SpringSecurity 6 的版本了，JDK 也强制要求至少是 17 版本。

至于为什么突然读了这本书，是因为 iw-mixes 项目做登录认证功能时，想到了 SpringSecurity ，但我又不会（可悲～）。新项目必定是用新版本，所以我就结合 SpringSecurity 6 源码读完了这本书，其中对我来说最大的区别可能就是 WebSecurityConfigurerAdapter 没有了，取而代之的是通过 @Scope("prototype") 注入 HttpSecurity Bean，再生成自己的 Security Configurer Builder 。

书中主要是以 Demo 的形式讲解 SpringSecurity 的用法，我主要是了解 SecurityFilterChain 的执行原理以及认证流程。

工作

一如既往的无聊，工作内容没啥兴趣。公司组织架构做了小调整，又参与了一个新项目的迭代，但对我来说好像整体上也没啥影响。

拖欠工资不发，真难受，必须要换工作了！房租要交不起了-_-

生活记录

基本上每天都会刷一刷 V2EX 、Feedly，时不时看一下 GitHub Trending、掘金、微信公众号。

记录一句从阮一峰周刊看到的别人写的话：一周是一年的2%。阮大神说挺震撼的，我看到后也是如此，平常感觉一周真的好快，感觉啥也没干，但就是这样的一周，一年只有50次左右。所以就算每周写周报，一年也就50篇左右，不是什么特别大的数字。既然如此，为何不坚持坚持呢，每周写一写，做个小总结，提醒自己不犯懒，让生活更加充实。

月末两个星期的双休，又连续跑了两次高速，“成功”在3个月磨合期内跑到了5000km，油费真贵！后悔没考虑电车了。
